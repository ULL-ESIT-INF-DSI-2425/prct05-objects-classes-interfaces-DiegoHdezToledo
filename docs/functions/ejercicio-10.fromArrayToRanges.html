Function fromArrayToRanges

  • fromArrayToRanges(numeros: number[]): string

    Converts an array of numbers into a string representation of ranges. Consecutive numbers are represented as a range (e.g., "1_3" for [1, 2, 3]). Non-consecutive numbers are represented individually.

    Parameters

    • numeros: number[]

      The array of numbers to be converted into ranges.

    Returns string

    A string representation of the ranges.

    Example

    fromArrayToRanges([1, 2, 3, 5, 6, 8]);
    // Returns "1_3,5_6,8"
